# Listas de Bloqueo DNS — Ciberseguridad Colombiana

Listas de bloqueo DNS recopiladas a partir de alertas oficiales de ciberseguridad colombiana, principalmente de:

- **CC-CSIRT Policía Nacional:** Centro Cibernético Policial  
- **ColCERT:** Grupo de Respuesta a Emergencias Cibernéticas de Colombia

Compatibles con **AdGuard Home**, **Pi-hole** y cualquier solución DNS que soporte el formato `0.0.0.0 dominio`.
